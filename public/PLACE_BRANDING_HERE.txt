BRANDING ASSETS REQUIRED
========================

Please add the following image files to this /public folder:

1. branding.png
   - The full TURSHIA.BOT branding image
   - Shows the logo, Бай Тошо character, and product jars
   - Recommended size: 1200x1600px or similar aspect ratio
   - Used in: Header logo display, social media sharing

2. logo.png
   - Square cropped version of just the TURSHIA.BOT wooden sign logo
   - Recommended size: 512x512px
   - Used in: Browser favicon (tab icon)

Where to find these images:
---------------------------
The branding image was provided in the project specification.

To prepare the assets:
1. Save the full branding image as "branding.png"
2. Crop just the logo area (wooden sign with text) and save as "logo.png"
3. Place both files in this /public folder

For detailed branding guidelines, see:
/BRANDING.md

After adding the files:
-----------------------
- Delete this PLACE_BRANDING_HERE.txt file
- Run `npm run dev` to see the branding in action
- The logo will appear in the header
- The favicon will show in the browser tab
